### **Local Development**
To build the CSS locally (optional):
```bash
npm install
npm run build
```

or, for development:
```
npm run dev
```

### Automated Build with GitHub Actions

This project uses GitHub Actions to:

- Generate the production CSS
- Output to dist/output.css
- Deploy automatically on push
- No manual build is required when deploying.
